# Student-Contact-Manager
 A beginner-friendly Python console application that helps a school administrator store and manage contact information for students, parents, and teachers.

The program stores contact information using Python dictionaries and prevents duplicate email addresses and phone numbers using sets.

Features

- Add new contacts
- View all contacts
- Search for a contact
- Update contact information
- Delete contacts
- Prevent duplicate email addresses
- Prevent duplicate phone numbers
- Validate user input with friendly messages

You can use Python 3 to run it and there is no need for any external Libraries
